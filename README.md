- http://<IP>:80/api/v2/status

- http://<IP>:80/api/v2/configurations/< config > (Nur mit Token-Authentifizierung), < config > =

  - NVM_PfcFixedCosPhi
  - NVM_PfcIsFixedCosPhiActive
  - NVM_PfcIsFixedCosPhiLagging
  - EM_ToU_Schedule
